阿里云[推荐]：https://mirrors.aliyun.com/pypi/simple/

临时使用：

可以在使用pip的时候加参数-i https://pypi.tuna.tsinghua.edu.cn/simple

例如：pip install matplotlib -i https://pypi.tuna.tsinghua.edu.cn/simple

这样就会从清华这边的镜像去安装matplotlib库。
